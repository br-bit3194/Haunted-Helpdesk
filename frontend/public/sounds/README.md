# Scary Audio Files

This directory should contain the following audio files for the 3D model scary effects:

## Required Files

1. **ambient_hover.mp3** - Scary ambient sound that plays when hovering over the model
   - Suggested: Low growl, hiss, or eerie whisper
   - Volume: 0.5 (moderate)
   - Loop: false

2. **scream_click.mp3** - Loud scary sound that plays when clicking the model
   - Suggested: Scream, roar, or jump-scare sound
   - Volume: 0.7 (loud)
   - Loop: false

3. **growl.mp3** - Alternative hover sound
   - Suggested: Deep growl or threatening sound
   - Volume: 0.5 (moderate)
   - Loop: false

4. **whisper.mp3** - Ambient background sound (optional)
   - Suggested: Creepy whispers or ambient horror sounds
   - Volume: 0.3 (quiet)
   - Loop: true

## Audio Format

- Format: MP3 (for broad browser compatibility)
- Bitrate: 128kbps recommended (balance between quality and file size)
- Duration: 1-3 seconds for effects, longer for ambient loops

## Sources

You can find free scary sound effects from:
- Freesound.org
- Zapsplat.com
- Mixkit.co
- YouTube Audio Library

## Note

The application will gracefully handle missing audio files and continue to work without sound if files are not present.
